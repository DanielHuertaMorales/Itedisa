
# Sitio Web ITEDISA

Este sitio está diseñado con HTML, TailwindCSS, PHP y MySQL.

- `index.html` muestra la descripcion general de quienes somos y que ofrecemos.
- `productos.php` carga productos por categoría.
- `marca.php` carga productos por marca.
- `contacto.php` carga un formulario de contacto para el usuario.

Para probar localmente, usa XAMPP o cualquier servidor PHP y crea una base de datos llamada `itedisa_bd`.
